
# Contributing to UK Passport Photo Master

Thank you for your interest in improving this project!

## How to Contribute
1. **Fork the Repository**: Create your own copy of the project.
2. **Create a Feature Branch**: Work on your changes in a dedicated branch.
3. **Write Clear Commit Messages**: Describe exactly what you fixed or added.
4. **Submit a Pull Request**: Provide a detailed description of your changes.

## Bug Reports
Please provide a clear description of the issue, including steps to reproduce and screenshots of the behavior.

## Code Standards
- Ensure all code is written in TypeScript.
- Follow the existing Tailwind CSS patterns for styling.
- Ensure any AI prompt changes maintain UK Passport compliance as the priority.
